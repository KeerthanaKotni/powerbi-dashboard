# Power BI Dashboard Project

## Overview
This project contains an interactive Power BI dashboard created for data analysis and visualization. The dashboard provides insights using charts, graphs, filters, and KPIs.

---

## Dashboard Features
- Interactive visualizations
- Data filtering and slicing
- KPI analysis
- Trend analysis
- User-friendly dashboard design

---

## Tools & Technologies Used
- Microsoft Power BI
- CSV Dataset
- GitHub
